Introduction to web design

What is web design?

Web design is the art of creating content for the internet. In this Coder Dojo course you will create your own personal web site that you can add to the internet and show to your friends. To create a web site you will need to know both HTML and CSS. HTML and CSS are the language of the World Wide Web and browsers, such as Google Chrome, Safari, Firefox and Internet Explorer use it to display the pages you see when you surf the web.

What is HTML and CSS?

When building a web page you will need to create 2 files. Firstly you will need to create a HTML file and this will contain all the text and data to be displayed on the web page. If you are going to create a web page about your favourite food, then you would probably want to have some paragraphs describing the food, a list of the ingredients, some photographs and perhaps a Youtube video of how you made it. All of this information will need to be added to your HTML file using tags to structure it in a way that a browser can understand.

Once you have all the information saved in your HTML file you can open and read it in your web browser. At this point it will just be black text on a white background. But that doesn't look very pretty. To personalise your web page with colours and fonts you will need to add a CSS stylesheet. You will then need to create a single CSS file and link that to all your HTML web pages so that your entire web site has a consistent theme.
